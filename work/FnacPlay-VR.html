FnacPLAY VR

CINEVR is an experience which allows you to play videos in a virtual theater with Virtual Reality headset support. It has spatialized sound support, dynamic lighting and high performance.

On this project I have worked on a lot of different things such as the implementation of a multiplayer network with Photon PUN and Photon Voice, some UI development such as an optimized system of scroll with pool for example, shaders and web API calls, etc. But also the realization of 3D and 2D assets in 3D Coat, Modo and Substance Painter.

  • Gear VR Store
  • VR
  • Unity 5
  • Gear VR
  • Cardboard

The scene is optimized for mobile rendering with a very low polygons count and a unique Atlas texture.
