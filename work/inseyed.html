InsEyed

- Developer

Each section of this application explains the functioning of a human eye and how some dysfunctions are resulting with a visual impairment.

InsEyed was made with the zSpace SDK available for Unity, which is handling the stylus controller and the stereoscopic view. On this project I have programmed in C# and also participated in the UX design.

  • AR
  • Unity 5
  • zSpace

With a separated view, the user can see how the far-sightedness is perceived. This is created with a second camera which is inside the 3D eye.

On this section all the anatomy of an eye is explained with the highlight of the selected composant.
