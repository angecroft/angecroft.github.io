FnacPLAY is one of the first VR application available on both Gear VR and Cardboard. You can watch your favorites VOD in a cinema theater environment. It has spatialized sound support, dynamic lighting and high performance.

On this project I have worked on the UI but also on the web FnacPLAY API calls and errors management witch were received as popup by the user.

  • AR
  • Unity 5
  • zSpace

Once the user is logged in, he has the choice to scroll its list of rental or bought movies and series.

On a movie detail page, all informations given by the FnacPLAY API is displayed such as the user rating, the description, duration, etc.
